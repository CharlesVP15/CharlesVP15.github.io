IDENTIFICADORES


      DEFINICIÓN:

Un identificador es el nombre que le damos a las variables y funciones, puede estar formado por una secuencia de letras y dígitos del 0 al 9, o que también acepta el caractér de subrayado. Un identificador no permite "tilde ( ´ )" ni "eñe ( ñ )".

Un identificador debe de cumplir las siguientes condiciones:

  1. El primer caractér debe ser una letra o el caractér del subrayado ( _ ).
  2. Los caracteres significativos son los 32 primeros del identificador.
  3. Se diferencian de las mayúsculas de la minúsculas.
      EJEMPLO:
Válidos No válidos
_num 1 not
nom dirección
lug_nac año_lec

VARIABLES


      DEFINICIÓN:

Las variables se utilizan para almacenar valores que pueden modificarse durante la ejecución de un programa. Antes de declarar una variables en primer lugar debe plantearse que tipo de datos quiere almacenarse. Ejm.:

                  float precio;
                  int n, a, z;
                  char *nom, *ape, *dir;
                  char nom[20], ape[20], dir[20];

      CLASIFICACIÓN:

VARIABLES LOCALES: Se declaran después de la función main() o en cualquier otra función en la que será utilizada.

VARIABLES GLOBALES: Se declaran antes de la función main(), se pueden usar en cualquier parte del programa y se destruye al utilizar esto.

CONSTANTES


      DEFINICIÓN:

Al contrario de las variables, las constantes son aquellos datos que mantienen su valor a lo largo de la ejecución de un programa. Para definir una constante con nombre utilice la directiva "#define".
